# 🛒 Retail Sales Analytics Dashboard

## 📌 Project Overview

This project analyzes the Brazilian Olist E-Commerce dataset using Python, Pandas, Matplotlib, and Streamlit.

The dashboard provides interactive insights into sales performance, customer behavior, revenue trends, product categories, and customer ratings.

---

## 🚀 Features

- Interactive Streamlit Dashboard
- Dataset Preview
- Key Metrics
- Monthly Revenue Trend
- Revenue by State
- Top 10 Product Categories
- Top 10 Customers
- Average Customer Rating
- Revenue by Order Status
- Sidebar Filters
- Download Filtered Dataset

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Streamlit
- Jupyter Notebook

```bash

```
